## 🚀 Key Highlights of the Portfolio App
This interactive portfolio is structured with clean, modular, and performant code. It features:

*   📂 **Multi-Tab Profile Deck**: Switching between:
    *   **Overview**: Executive summary, career objective, and contact quick links.
    *   **Education & Experience**: Timelines mapping academic scores and internships.
    *   **Skills Matrix**: Detailed ratings of various technical proficiencies.

    *   **Credentials & Achievements**: Certified courseworks (NPTEL, Infosys Springboard) and athletic trophies.

*   🌌 **Cosmic Dark Theme**: A high-contrast premium dark workspace, styled elegantly using Tailwind CSS, pairing modern display typography with interactive hover and focus animation states.

## 💻 Running Locally

### Prerequisites
Ensure you have Node.js (v18+) installed.

```bash
# Clone this custom repository
git clone https://github.com/Mounika-420/Mounika-Selvaraj-Portfolio.git

# Jump into project directory
cd Mounika-Selvaraj-Portfolio

# Install dependencies 
npm install

# Run Vite-Express local server
npm run dev
```

Open http://localhost:3000 to view your local app!
