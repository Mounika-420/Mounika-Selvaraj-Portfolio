# Mounika Selvaraj - Personal Portfolio & Interactive Resume 🚀

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://mounika-selvaraj-portfolio.vercel.app/)

Welcome to the repository of my interactive personal portfolio website! This modern, highly responsive web application serves as my digital resume, showcasing my professional details, educational history, software development skills, and technical projects.

Live Site: [https://mounika-selvaraj-portfolio.vercel.app/](https://mounika-selvaraj-portfolio.vercel.app/)

Built with a blend of **React, TypeScript, Tailwind CSS**, and structured to run seamlessly on both development environments and production, this portfolio highlights my journey as a Computer Science & Business Systems student at **V.S.B. Engineering College**.

---

## 🌟 Key Features

- **Interactive Profile Tab System**: Seamlessly switch between sections:
  - **Overview**: A professional summary, career objective, and quick profile highlights.
  - **Education & Experience**: Detailed timeline of academic achievements and internship experiences.
  - **Skills Matrix**: Visual representation of language, backend, frontend, and database proficiencies.
  - **Projects Showcase**: Interactive view of fully functional applications with details on technology stacks.
  - **Credentials**: Showcase of industry-recognized certifications (Infosys Springboard, NPTEL) and athletic/coding achievements.
  - **Contact**: An easy-to-use form and verified contact channels.
- **Printable Resume Modal**: An elegant, perfectly formatted, single-page print layout to export/download a clean PDF resume.
- **Modern Dark & Light Accents**: Crafted with a premium dark visual palette using sleek typography and smooth layout animations.
- **Fully Responsive**: Optimized for desktops, tablets, and mobile screens.

---

## 🛠️ Tech Stack & Technologies

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, Lucide React (Icons)
- **Animations**: Motion (`motion/react`)
- **Backend & Serving**: Express, tsx, esbuild (optimized for lightweight runtime and high-performance server-side deployment)
